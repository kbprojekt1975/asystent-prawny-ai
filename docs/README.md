# Dokumentacja Systemu Asystent Prawny AI

Witaj w pełnej dokumentacji technicznej projektu **Asystent Prawny AI**. Ten dokument służy jako punkt wejścia dla programistów seniorów i architektów chcących zrozumieć, rozbudować lub wdrożyć system.

## Spis treści

1.  [**Architektura i Stos Technologiczny**](01-architecture.md)
    *   Przegląd wysokopoziomowy, stos technologiczny, przepływ danych.
2.  [**Frontend (React + Vite)**](02-frontend.md)
    *   Struktura komponentów, zarządzanie stanem (hooki), system stylizacji.
3.  [**Backend (Cloud Functions)**](03-backend.md)
    *   Logika serwerowa, integracja z Gemini AI, zarządzanie sekretami.
4.  [**Baza Danych i Model Danych**](04-database.md)
    *   Schema Firestore, struktura dokumentów, zasady bezpieczeństwa.
5.  [**Podręcznik Wdrożenia i Rozwoju**](05-dev-ops.md)
    *   Lokalne środowisko, emulatory, proces CI/CD, secrety.
6.  [**Konfiguracja Ngrok**](06-ngrok-setup.md)
    *   Tunelowanie dla testów mobilnych, konfiguracja HTTPS.
7.  [**Wielojęzyczność i Lokalizacja (i18n)**](07-i18n.md)
    *   System tłumaczeń, wsparcie PL/EN/ES, wielojęzyczne prompty AI.
8.  [**Responsywność i Mobile-First Design**](08-responsive-design.md)
    *   Breakpointy, optymalizacje mobile, grid layouts, najlepsze praktyki.
